dva框架搭建的用户管理demo

在本地run起来前提条件：
1. 确保 node 版本是 6.5 +
2. 用 cnpm 或 yarn 能节约你安装依赖的时间

再执行以下操作
1. git clone http://54.223.153.229/mengying/dva-users.git
2. cd dva-users
3. npm install 
4. npm run build
5. npm start 即可看到效果

如果想自动生成dva的其他路由，model或component等基础文件，还需要安装0.7及以上版本的dva-cli:
	$ npm i dva-cli -g
	$ npm -v
